Что такое ссылка?

--> Вниз

Ссылка обычно выглядит как подчёркнутый участок текста, щёлкая на который вы переходите на другую страницу,
открываете изображение или начинаете скачивать файл. Если представить, что интернет это огромная сеть из множества узлов,
то ссылки будут нитками, соединяющими все узлы этой сети.

Ссылки создаются с помощью очень простого и короткого тега <a>.

Адрес ссылки задаётся с помощью атрибута href
"Название" ссылки пишется внутри тега <a> </a>

Для того, чтобы добавить ссылке всплывающую подсказку, которая показывается если задержать мышку на ссылке, надо использовать атрибут title

добавить картинку на страницу

чтобы добавить на страницу изображение, нужно использовать одиночный тег <img> с атрибутом src, в котором указан адрес картинки
Чтобы управлять шириной или высотой изображения, нужно использовать атрибуты width и height. можно задать размеры в %
Если вы задаёте размер картинки в пикселях, то используйте просто цифры. Добавлять px не нужно, таков стандарт.
Если задать только один из размеров, ширину или высоту, то вторую размерность браузер вычислит самостоятельно исходя из пропорций изображения.
Если же задать и ширину, и высоту для картинки, То браузер может нарушить пропорции исходного изображения.
<img src="img1.jpg" alt="тут фотка кошака">
так же можно указать "подсказку" атрибутом title

текст который выводится если не прогрузилась картинка указываем в атрибуте alt

Ссылка с якорем

Ссылки с якорем обычно используются для создания навигации внутри страницы. Например, оглавления в начале страницы с большой статьёй.
Ссылка с якорем содержит символ #, после которого идёт идентификатор. Идентификатор создаётся с помощью атрибута id, который может быть задан у любого тега.
Можно задать адрес, состоящий из одного якоря, например: <a href="#begin">Начало</a>
При щелчке на такую ссылку браузер найдёт на странице элемент с атрибутом id со значением begin и прокрутит окно страницы к нему. То есть перезагрузки страницы не произойдёт
Якорь можно использовать и в абсолютных адресах, тогда после перехода на нужную страницу произойдёт прокрутка к заданной части этой страницы.

Изображение-ссылка

Ссылки можно делать не только с помощью текста, но и с помощью изображений. Для этого нужно обернуть тег img в тег a
