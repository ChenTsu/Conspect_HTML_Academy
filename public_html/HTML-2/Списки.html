Списки

Упорядоченный (автоматически нумерующийся) список создается при помощи тега <ol></ol>
элементы списка обрамляются в тег <li></li>
атрибут start тега <ol></ol> используется для задания начального значения "счетчика"
атрибут type тега <ol></ol> используется для задания типа нумерации списка:

  1. type="1" - Арабские числа
  2. type="A" - Прописные буквы латинского алфавита
  3. type="a" - Строчные буквы латинского алфавита
  4. type="I" - Римские числа в верхнем регистре
  5. type="i" - Римские числа в нижнем регистре

"неупорядоченный" (маркированный) список создается при помощи тега <ul></ul>
Маркеры могут принимать один из трех видов: круг (по умолчанию), окружность и квадрат
Для выбора стиля маркера используется атрибут type тега <ul>
Допустимые значения type : disc, circle, square

  • элементы списка добавляются так же
  • тегом <li></li>

Многоуровневый список делается вложением еще одной пары тегов списка <ol></ol> или <ul></ul> в элемент <li></li> существующего списка

Список определений

Список определений создаётся с помощью трёх тегов:
<dl> обозначает сам список определений;
&tl;dt> обозначает термин;
&tl;dd> обозначает определение термина.
Теги <dt> и <dd> пишутся парами внутри <dl>.
